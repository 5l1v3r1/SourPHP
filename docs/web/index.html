SourPHP::Ek$iSözlük PHP İstemci Kütüphanesi

Nedir?

SourPHP, Ekşisözlük'deki bazı verileri alabileceğiniz bir PHP kütüphanesidir.

Nerelerde kullanılabilir

SourPHP, takip edilen başlıklarda bir değişiklik olup olmadığını kontrol etmek amacıyla geliştirilmeye başlandı. Yani, "şu başlıkta değişiklik olmuş mu?" gibi kontrolleri yapmak için kullanılabilir.

Edinme

Kodları edinmek için github üzerindeki http://github.com/yuxel/SourPHP/tree/ adresini ziyaret edebilirsiniz.
Örnek kodları ve kütüphane kodlarını git clone git://github.com/yuxel/SourPHP.git ile kendi deponuza çekebilirsiniz.
Ek: Kodları SourceForge sunucularından da çekebilirsiniz.
